Chowdren is a blazingly fast runtime for Clickteam Fusion.

It's cross-platform, running under desktop, mobile and console platforms.

FEATURES

  • Fast, portable runtime for Clickteam Fusion
  • Translates your events to optimized C++ code for amazing performance
    (~8x times faster)
  • Runs on Windows, Linux, Mac OS X, Android, Wii U, 3DS, PS4 and PS Vita
  • Has a big library of more than 65 extensions
  • Uses GPU hardware acceleration, with 31 shaders and effects already ported
  • Merges together identical events and sprites for smaller binaries
  • Dual-licensed under the GPL and a commercial license
